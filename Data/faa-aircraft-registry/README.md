# FAA Aircraft Registry Dataset

This directory contains technical aircraft registration data used for asset tracking.

### Contents:
- **ACFTREF.txt**: Aircraft Reference file, providing technical specifications for different aircraft models. Used for `Dim_Aircraft`.
- **MASTER.txt**: Aircraft Registration Master file, providing registration details, owner information, and manufacture year for specific aircraft (linked by N-Number).
- **ardata.pdf**: Technical documentation provided by the FAA explaining the schema of the registry files.

### Source:
[FAA Releasable Aircraft Database](https://www.faa.gov/licenses_certificates/aircraft_certification/aircraft_registry/releasable_aircraft_download)
